The Oath of Standards

[as of 24 November 2013 - see tmp.com for full documentation]

The members of the TMP UI Super Friends pledge to support:

Code and accessibility standards

  • World Wide Web Consortium (W3C) best practices
  • WCAG 2.0 Level A success criteria

Browsers

For all browsers except Internet Explorer, our company supports the current and prior major release versions. Supported versions of Internet Explorer are as listed below. Modified versions and forced "Compatibility Mode" are not supported.

  • Chrome
  • Firefox
  • Safari
  • Internet Explorer 10
  • Internet Explorer 9

The Super Friends will also offer limited support to Internet Explorer 8 for candidate-facing websites.
